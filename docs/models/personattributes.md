# PersonAttributes

A person is a record of someone known to DailyPay. There will only ever be one person record per human being.

## Example Usage

```typescript
import { PersonAttributes } from "dailypay/dailypay";

let value: PersonAttributes = {
  disallowReason: null,
  stateOfResidence: "NY",
};
```
